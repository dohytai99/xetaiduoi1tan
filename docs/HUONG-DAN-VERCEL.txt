ĐƯA SITE TĨNH LÊN VERCEL
=======================

Thư mục có index.html ở gốc — Vercel nhận là site tĩnh (HTML + CSS).

Cách A — Qua web
----------------
1. https://vercel.com → đăng nhập.
2. Add New… → Project → Import Git Repository (push repo lên GitHub trước).
3. Framework: **Other** — Root: gốc repo.
4. Build / Output: **để trống**.
5. Deploy → mở URL xxx.vercel.app.

Cách B — CLI
------------
cd thư mục dự án → vercel → (lần sau) vercel --prod

Ghi chú: Cài đặt → Domains trên Vercel khi gắn tên miền riêng. Chọn **một** domain chính (www hoặc không www) trong Domains → “Redirect to…” để tránh vòng redirect nếu vừa cấu hình Vercel vừa thêm rule trùng trong `vercel.json`.

Web Analytics (không thấy dữ liệu?)
-------------------------------
- Trong Vercel: Project → Analytics → bật **Enable** (sau đó deploy lại).
- Đường dẫn `/_vercel/insights/script.js` chỉ do **Vercel** phục vụ. Nếu site chỉ nằm trên **GitHub Pages** / mở file `file://` / host khác → script 404, dashboard Vercel **không** có pageview.
- Kiểm tra: mở site bằng URL **xxx.vercel.app** (hoặc domain đã trỏ DNS về Vercel), F12 → Network → tải được `script.js` và không bị extension chặn.
